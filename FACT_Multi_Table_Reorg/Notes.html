Reorganization for Database Tables

The FACT "New Multi Table Reorg Job" wizard prepares FACT, CoSort and bulk DB load scripts for running external table reorganizations (i.e. unload, sort, load) in one operation. The wizard allows the fast execution of multi-table reorgs in this way, and prefaces future IRI ETL and FieldShield operations where multiple tables are selected at once, and big data movement operations are combined in one pass.

How to run a Reorg Demo:

  1. Select the FACT_Multi_Table_Reorg project in the Project Explorer view.
  2. In the Data Source Explorer, select Oracle -> XE -> Schemas -> SCOTT.
  3. Open the "New Multi Table Reorg Job" wizard from the FACT icon.
  4. Select the loader type Oracle. Click Next.
  5. On the Data Extraction Stage page, select tables (don't use the FACT-ETL demo 'ORDERS' table). Click Next. The Field Modification Rules opens. This demo requires no modifications, so click Next.
  6. On the Data Loading Stage page, in the Connection profile field, select the loader target database Oracle.
  7. Select Truncate tables before insert if you are going to reorg existing tables. Click Next.
  8. Review the reorg parameters on the Summary page. Click Finish.
  9. Open the demo project explorer folder to examine all the files created. Any of these are available for editing, especially the SortCL scripts, for additional data transformation, reporting and/or protection /OUTFILE layouts in addition to the load target.
  10. Right-click on the batch file and select Run As -> Batch Program. To configure, run, or schedule the batch file, right-click on the batch file and select Run As -> External Tools Configuration.
  11. Verify sorted results in target database tables in the Data Source Explorer.
