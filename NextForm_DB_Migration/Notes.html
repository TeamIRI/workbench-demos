Migration of Database Table

The NextForm New Single Source Migration Job; wizard migrates a single source (file or table) to another target.

IMPORTANT: Before running this demo, expand the setup folder and right-click reset_demo.bat. Select Run As > Batch Program.

Recommended steps to perform this demo in IRI Workbench:

  1. Select the NextForm_DB_Migration project in the Project Explorer view.
  2. In the Data Source Explorer, select Oracle > XE > Schemas > SCOTT > COMPANIES_TEST.
  3. Click Next Form menu and New Single Source Migration Job.
  4. Enter companies as the file name. Click Next.
  5. Click the Browse button in Metadata and select Companies.ddf and OK. When prompted, click Yes to copy the fields to the job.
  6. Click Create New Table. Select MySQL for the DSN. In the textbox, enter test.COMPANIES_TEST. Click OK.
  7. Click Next and Next again.
  8. Enter 19 next to COMPANY_NUMBER and enter 50 next to COMPANY_NAME. Click Next.
  9. Under Primary Key Name, enter Customers_Test_PK. Check COMPANY_NUMBER Click Finish
  10. Run the job by creating the table first. Open test_COMPANIES_TEST.sql. Under Name, select MySQL and under Database, select test. Right-click in the script and select Execute All.
  11. Right-click companies.ncl and select Run As > IRI Job.
