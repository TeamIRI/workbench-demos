NextForm Data Federation

This demonstrates the use of the NextForm (NF) new reformat job wizard to combine specific information found in two different sources through the use of a pre-action (INREC). The sources are a database table and a delimited file. The results are displayed on the screen.

This demo uses the IRI Workbench.

Recommended steps to perform the demo in the Workbench:

  1. Open the "New Reformat Job wizard from the NextForm" toolbar menu.
  2. Select the project and name the job file datafederation.
  3. Click Next to go to the Data Sources page.
  4. Click Add Data Source on the right to open the Data Source dialog screen.
  5. Select ODBC and click Browse. The ODBC Table Selection dialog screen opens.
  6. In the Data source name (DSN) field, select Oracle.
  7. In the Table name field, select JOB_HISTORY. Click OK.
  8. Click OK on the Data Source dialog.
  9. Click Add Existing Metadata on the right. Select historytable.ddf and click OK.
  10. When the Save specification message opens, select Yes.
  11. Click Add Data Source on the right to open the Data Source dialog screen to add the second source file.
  12. Select Files and enter the file name jobs.txt.
  13. Click OK on the Data Source dialog.
  14. Click Edit Source Options on the right. The Source Options page opens.
  15. In the Format field, select DELIMITED. Click OK.
  16. Click Add Existing Metadata on the right, and then select jobs.ddf. Click OK.
  17. When the Save specification message opens, select Yes.
  18. Click Pre-Action Layout on the right. The Pre-Action page opens.
  19. Click Edit Pre-Action Layout on the right. The Pre-Action Field Layout screen opens.
  20. Click the Fixed/Delimiter button. Select Fixed and OK.
  21. Double-click on the JOB_ID row. The Pre-Action Field editor opens.
  22. In the Size field, change the size to 10.
  23. Click OK.
  24. On the Pre-Action Field Layout page, click OK.
  25. Back on the Pre-Action page, click Next. The Data Targets page opens.
  26. Click Add Data Target on the right to open the Data Target dialog screen.
  27. Select Standard out. Click OK.
  28. To create a file in addition to the screen view of standard out, click Add Data Target on the right to open the Data Target dialog screen again.
  29. Select Files and enter the new file name jobs_id.out. Click OK.
  30. On the Data Targets page, click Finish.
  31. Run the script by right-clicking in the script and selecting Run As -> IRI Job.
