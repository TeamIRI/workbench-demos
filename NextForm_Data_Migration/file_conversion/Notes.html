NextForm File Conversion

This demonstrates the use of the NextForm (NF) new reformat job wizard to convert a delimited flat file to a fixed file.

This demo uses the IRI Workbench.

Recommended steps to perform the demo in the Workbench:

  1. Open the New Reformat Job wizard from the NextForm toolbar menu.
  2. Select the project and name the job file delimited_to_fixed.
  3. Click Next to go to the Data Sources page.
  4. Click Add Data Source.
  5. In the Files field, click Browse to find and select chiefs.sep. Click OK.
  6. Click Add Existing Metadata. Select chiefs.ddf and click OK. The following message opens:
    Would you like to copy the fields in the DDF to the job section? If No is selected then a /SPECIFICATION with a reference to the DDF will be added. Click No.
  7. Click Edit Source Options to open the Source Options screen.
  8. In the Format field, open the drop-down menu and select DELIMITED. Click OK.
  9. Click Next to progress to the Data Targets page.
  10. Click Add Data Target to open the Data Target screen.
  11. In the Files field, enter the name of the outfile, fixed_file. Click OK.
  12. Click Edit Target Options.
  13. In the Format field, open the drop-down menu and select FIXED.
  14. In the Header Record field, enter the header for the fixed file including the double quotes:
    "President <13 spaces> Votes <2 spaces> Service <6 spaces> Party <1 space> St"
  15. Click OK.
  16. Click Target Field Layout on the Data Targets page.
  17. Click the Fixed/Delim icon in the Targets section.
  18. In the Type field, select Fixed. Click OK.
  19. Click the Position and Size values to enter the following values for each field:
    pres - position=1, size=22
    votes - position=23, size=5
    service - position=30, size=12
    party - position=43, size=5
    state - position=49, size=3
  20. Click OK.
  21. Click Finish.
  22. In the Editor view, edit the header by adding a space and \n\n after St, before the closing quotation mark. This adds two line feeds to the header.
  23. Run the script by right-clicking in the script and selecting Run As > IRI Job.

You can also convert ascii to unicode. See the script named ascii_to_unicode.scl.
